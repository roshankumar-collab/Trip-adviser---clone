# TripAdvisor Clone 🌍

## Project Overview
This project is a front-end clone of the TripAdvisor website built using HTML and CSS. It is designed to replicate the look and feel of the original platform while adding creative improvements like modern UI sections, animations, popups, icons, and an attractive footer.

## Features
- Responsive Homepage
- Navigation Bar
- Search Section
- Hotels Recommendations
- Things To Do Section
- Restaurant Listings
- Destination Cards
- Interactive Popups
- Hover Effects and Animations
- Icons for Better UI
- Stylish Footer
- User-Friendly Layout

## Technologies Used
- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

## Project Purpose
The purpose of this project is to practice front-end development by recreating a popular travel website interface and improving it with unique design elements.

## Pages Included
- Home Page
- Hotels Section
- Things To Do Section
- Restaurant Section
- Contact/Footer Section

- Clean and structured layout
